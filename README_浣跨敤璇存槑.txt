林海回声：东北虎数据新闻交互网页（8页左右并列双页版）

使用方法：
1. 保持整个文件夹结构不变，双击 index.html，建议使用 Chrome 或 Edge 浏览器打开。
2. 封面为单页大图；进入正文后为左右并列的纯色日记页。
3. 鼠标滚轮用于本页内容滚动；翻页使用左侧目录、键盘左右方向键或触屏左右滑动。
4. 内容图片可点击放大；文艺作品和政策资料可使用左右按钮滑动浏览。
5. 尾页点击“开始游戏”进入独立全屏游戏；Esc 或右上角 × 返回日记。

本版主要调整：
- 页面由14页合并为8页，每个小节完整呈现后再翻页。
- 网页背景使用 assets/新背景.png；日记内页改为纯色，去掉横线与图片边框。
- 正文统一使用宋体、首行缩进并放大；所有数字使用“潮字社混元简繁.ttf”。
- 引言使用“老虎素材2.png”和“数据板.png”重新排版，保留数字增长动画。
- 全球地图国家介绍改为原页点击切换，国家点位重新调整。
- 盗猎去向使用按钮切换完整文字；印度案例使用便签式排版。
- 中国方案使用 assets/流程图.png 制作六步自动播放、可点击切换的预警流程动画。
- 弹幕使用“东北虎相关滚动弹幕1000条.txt”生成的 barrage-data.js，完整句子慢速从右向左循环。
- 游戏使用老虎素材3、小猪、小鹿、公路、偷猎、房屋和监管等新图标。

重要文件：
- index.html：页面结构与正文
- style.css：排版与视觉样式
- script.js：翻页、地图、图表、流程动画、轮播和游戏逻辑
- barrage-data.js：1000条弹幕的本地浏览器数据
